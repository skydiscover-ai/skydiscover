## Overview

SkyDiscover ships with a comprehensive benchmark suite spanning mathematical optimization, real-world systems, GPU kernels, competitive programming, reasoning, and creative tasks. Use these benchmarks to evaluate search algorithms, compare model providers, or learn from working examples.

## Quick Start

### Installation

```bash
uv sync                              # Base install
uv sync --extra math                 # Math benchmarks
uv sync --extra adrs                 # ADRS systems benchmarks
uv sync --extra frontier-cs          # Frontier-CS benchmark tooling
uv sync --extra prompt-optimization  # Prompt optimization
```

Combine extras as needed:

```bash
uv sync --extra math --extra adrs
```

### Running a Benchmark

```bash
uv run skydiscover-run \
  benchmarks/math/circle_packing/initial_program.py \
  benchmarks/math/circle_packing/evaluator.py \
  --config benchmarks/math/circle_packing/config.yaml \
  --search adaevolve \
  --iterations 100
```

## Benchmark Structure

Every benchmark follows the same layout:

```
benchmark_name/
├── initial_program.py    # Starting solution with EVOLVE-BLOCK markers
├── evaluator.py          # Scoring function (evaluate(program_path) -> dict)
├── config.yaml           # Algorithm and model configuration
└── README.md             # Problem description and references
```

- `initial_program.py` — The seed solution. Contains `# EVOLVE-BLOCK-START` and `# EVOLVE-BLOCK-END` markers around the region to mutate.
- `evaluator.py` — Must define `evaluate(program_path)` returning a dict with `combined_score`.
- `config.yaml` — Sets the LLM, search algorithm, and evaluator parameters.

## Creating Your Own Benchmark

1. **Write an evaluator** with `evaluate(program_path)` that returns `{"combined_score": ...}`.
2. **Write an initial program** (optional) with EVOLVE-BLOCK markers.
3. **Create a config YAML** with LLM and search settings.
4. **Run it:**

```bash
skydiscover-run my_benchmark/initial_program.py \
  my_benchmark/evaluator.py \
  -c my_benchmark/config.yaml \
  -s adaevolve -i 100
```

See the [Custom Problems guide](/examples/custom-problems) for detailed instructions.

## Performance Comparison

Across ~200 optimization benchmarks, AdaEvolve and EvoX achieve the strongest open-source results:

| Metric | Result |
|--------|--------|
| Frontier-CS (172 problems) | ~34% median score improvement over OpenEvolve, GEPA, ShinkaEvolve |
| Math + Systems (12 tasks) | Matches/exceeds AlphaEvolve and human SOTA on 12/14 tasks |
| Cross-cloud transfer cost | 41% lower (CloudCast) |
| GPU load balance | 14% better (EPLB) |
| KV-cache pressure | 29% lower (Prism) |
